开口即画。

声笔是一款纯语音控制的绘图工具——不用鼠标，不用键盘，说出指令即可创作。语音在浏览器识别，指令由 Python 后端解析执行，作品自动保存在服务器。

“画一个红色的大圆在左上角” “画三个并排的蓝色方块，然后把背景换成黑色” “把第二个方块变成黄色，再放大一点”
受浏览器安全策略限制，开启麦克风需要这一次点击；之后的所有操作均通过语音完成。建议使用 Chrome 或 Edge 浏览器，并允许麦克风权限。

指令手册

所有指令都支持自然语序与同义说法，可用「然后 / 接着 / 再」把多条指令连成一句话。

绘制

画一个〔颜色〕〔大小〕〔图形〕画一个红色的大圆
画 N 个…〔并排/竖排/随机〕画五个并排的绿色星星
…在〔位置〕在左上角画一个三角形
画一个空心的…画一个空心的蓝色方块
写上〔内容〕写上你好世界
再来一个复制上一个图形
